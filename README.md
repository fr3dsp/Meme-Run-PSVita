# Meme Run Decomp
Meme Run was a weird endless runner released exclusively on the Wii U until it eventually got pulled from the eShop due to copyright issues. Don't ask why I decompiled it, I got dared to do so `¯\_(ツ)_/¯`

So, it's a mostly functional decomp of Meme Run that also works in the editor with the controls mapped one to one, meaning if the game says A, B, X or Y is used, just press those letters on your keyboard. 
All sounds are present but they aren't the original ones, they've been recreated either by getting the original isolated sound or by isolating it as best as I could like illumconfirmed but they're not the same data as the .audioClips because AssetRipper can't extract those.

Both my version of the sounds and AssetRipper's .audioClip version are included, so if you manage to extract the sound's original data then make a pull request and I'll merge it
