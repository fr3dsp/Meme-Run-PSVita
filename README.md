# Meme Run Decomp
Yeah, Meme Run decompiled just because

So, it's a mostly functional decomp of Meme Run that also works in the editor with the controls mapped one to one, meaning if the game says A, B, X or Y is used, just press those letters on your keyboard. Also, it's missing 2 sound effects (ilumconfirmed & intervention 420) because AssetRipper can't decompile the GCADPCM format used in game to play audio on the gamepad, the rest of the sound effects are present becuase the music was ripped and preserved online and the others sfx were incredibly popular and common during the mlg era

That's it, if you manage to extract the sfx that are missing then make a pull request and I'll merge it
